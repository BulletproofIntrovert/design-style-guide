BADGE DESIGN CHALLENGE

What

freeCodeCamp needs help designing a set of badges for it's students as they complete different challenges in the program.

Why

freeCodeCamp wants to leverage more of its campsite theme to create a more consistent FCC brand. In order to do this we want to create FCC badges to help inspire and reward the coders going through our program!

Badges to be designed

  • Bear Attack Survivor
  • Byplane Crash Scavenger
  • Deep Cave Spelunker
  • Camp Crystal Lake Survivor
  • Poison Sucker-outer
  • Cave Painting Artist
  • Vine Swinger
  • Disentary Survivor
  • Forest Fire Fighter
  • Trench Foot
  • Aloe Vera Medic

Materials

We ask that you please use the FCC style guide for the colors.
